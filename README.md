# Worktime – Учёт рабочего времени

Веб-приложение для учёта рабочего времени сотрудников с мобильным интерфейсом и панелью администратора.

## Возможности

- Регистрация сотрудников по уникальной ссылке (отправляется администратором).
- Мобильный интерфейс с кнопками: "Работа начата", "Пауза", "Возобновить", "Закончить работу".
- Административная панель: управление сотрудниками, просмотр календаря, редактирование времени, отчёты с экспортом в Excel.
- Тёмная тема на всех страницах.

## Технологии

- Python 3, Flask, SQLAlchemy, Flask-Login
- HTML, CSS, JavaScript
- openpyxl (экспорт в Excel)
